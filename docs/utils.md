# WorldMonitor 추가 기능 목록

## 1. 지진/산불/기후 데이터 확장

### 1.1 새로운 재해 레이어 추가
- **화산 폭발**: USGS火山 데이터 API (https://volcano.si.edu/)
- **허리케인/태풍**: NOAA Hurricane 데이터 API (https://www.nhc.noaa.gov/)
- **홍수**: USGS Flood 데이터 API (https://water.usgs.gov/floods/)
- **토네이도**: NOAA Storm Data API (https://www storms.ncep.noaa.gov/)
- **쓰나미**: NOAA Pacific Tsunami Warning Center API
- **공기질/연기**: AirNow API (https://docs.airnowapi.org/) - 산불 연기와 공기질 정보
- **한파/폭염**: NOAA 열스트레스 데이터

### 1.2 지진 분석 강화
- **규모 분포 차트**: Magnitude별 막대 차트
- **깊이 분포**: Depth별 히스토그램
- **지역별 빈도**: 시간 경과에 따른 지역별 지진 발생 빈도
- **심도 vs 규모 상관관계**: 3D 산점도
- **Aftershock 패턴**: 여진 발생 패턴 분석

### 1.3 산불 분석 강화
- **FRP 차트**: 화방출률(FRP) 시간 경과 차트
- **연기 확산**: 대기 질 데이터와 결합한 연기 확산 시뮬레이션
- **역사적 비교**: 같은 지역의 역사적 산불 패턴 비교
- **위험도 예측**: 기상 데이터와 결합한 산불 위험도 예측

## 2. 지도 시각화 기능

### 2.1 새로운 시각화 레이어
- **히트맵**: 지진 규모/산불 FRP 강도 기반 히트맵
- **클러스터링**: 대용량 데이터 포인트 클러스터링
- **3D 막대 차트**: 지진 규모를 3D 막대로 표시
- **원형 레이어**: 진앙 크기를 원 크기로 표시
- **흐름 레이어**: 진동 전달 방향 표시

### 2.2 고급 지도 기능
- **시간 슬라이더**: 과거 데이터 타임라인 탐색
- **3D 건물/지형**: CesiumJS 기반 3D 지구본
- **커스텀 마커**: 재해 유형별 아이콘 마커
- **팝업/툴팁**: 마커 클릭 시 상세 정보
- **지정밀도 검색**: 좌표 기반 재해 검색
- **다중 레이어 오버레이**: 여러 재해 유형 동시 표시
- **지오폴리 경계**: 국가/지역 경계 표시
- **위험도 계층**: 재해 위험도 색상 계층
- **비교 모드**: 두 시기/지역 비교
- **지진파 애니메이션**: P파/S파 전파 애니메이션
